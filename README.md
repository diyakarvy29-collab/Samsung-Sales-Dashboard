# Samsung Sales Dashboard (Power BI)

## Overview
This project analyzes Samsung sales performance using an interactive Power BI dashboard.

## Key Insights
- Revenue declined around 2022 and recovered in 2024
- Galaxy S22 is the highest contributing product
- North America and Latin America generate the most revenue
- Sales trends show fluctuations indicating inconsistent growth
- YoY growth is significantly high, indicating a sharp increase

## Features
- KPI Cards (Revenue, Profit Margin, AOV, YoY Growth)
- Sales trend analysis over time
- Region-wise performance
- Product-wise contribution

## Files Included
- Power BI dashboard (.pbix)
- Project presentation (PPT/PDF)

## Tool Used
Power BI
